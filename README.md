# 💻 Web Development Workshop Registration Form

A clean and responsive **Web Development Workshop Registration Form** built using **HTML5** and **CSS3**. This project demonstrates modern **Flexbox layout**, **responsive web design**, **media queries**, **Font Awesome icons**, and a professional registration form UI.

## 📱 Responsive Design

This project is responsive across multiple screen sizes.

### 💻 Desktop

- Image and Form displayed side by side.

### 📱 Tablet

- Reduced image size.
- Container width adjusted.

### 📱 Mobile

- Image and form stack vertically.
- Inputs become wider.
- Heading text scales down.
- Layout adjusts automatically.

### 📱 Small Mobile

- Smaller fonts.
- Smaller image.
- Buttons stack vertically for better usability.
